
# Programa de Cálculo del IMC

## Descripción
Este programa calcula el Índice de Masa Corporal (IMC) utilizando una interfaz gráfica de usuario. Permite a los usuarios ingresar su peso, altura, sexo y edad para calcular su IMC y guardar estos datos en un archivo CSV.

## Instalación
1. Descargue el archivo `imc.exe`.

## Uso
1. Ejecute el archivo `imc.exe`.
2. Ingrese su nombre, peso (en kg), altura (en metros), sexo y edad.
3. Haga clic en el botón "Calcular" para ver su IMC calculado.
4. Haga clic en el botón "Guardar" para almacenar sus datos en un archivo CSV.
